# super_heroes

## Description
A Flask REST API that manages heroes, powers, and hero abilities.

## Setup
- Clone the repo
- Create and activate a virtual environment
- Install dependencies
- Run migrations
- Seed the database
- Start the server

## Endpoints
- GET /heroes
- GET /heroes/:id
- GET /powers
- GET /powers/:id
- POST /hero_powers

## Validations
- Power description must be at least 20 characters
- HeroPower strength must be Strong, Weak, or Average

## Testing
Use the provided Postman collection to test all endpoints.
